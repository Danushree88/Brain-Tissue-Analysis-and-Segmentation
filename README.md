# Brain-Tissue-Analysis-and-Segmentation
Brain tissue segmentation and disease support system using deep learning. U-Net trained on IBSR achieves GM Dice 0.914. Volumetric biomarkers (BPF, GM/WM ratio, CSF fraction) scored against healthy reference. Three transfer-learning models benchmarked on BraTS tumor classification - VGG16 achieves 94.3%.
